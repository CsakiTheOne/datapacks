Csáki's datapack collection

Go to GitHub to get some datapacks

Anvil magnet (in progress)

You can pull items from greater distances if you hold an anvil in your hand.
You can also put an anvil on top of hoppers if you use an item frame.
Known issues: only works with undamaged anvils

Fletching table functionality

Use the fletching table by throwing 2 sticks on top of it.
Works with hoppers too.

Mining plus

Mining related advancements.

Mixed stone tools

Craft stone tools with any variation of stone blocks.
Stone, cobblestone, granite, andersite, diorite.

Nitwit (in progress, just started)

Makes the nitwit villager useful.

Team wool

Makes team games possible in survival minecraft.
Put a colored wool in an item frame and step on it.
Red, yellow, (light) green, blue.

Void fog (in progress)

Brings back the void fog in modern versions of minecraft.

Woodcutter (in progress)

Makes the stonecutter to be able to cut logs.
